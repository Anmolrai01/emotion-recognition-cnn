# Emotion Recognition using CNN

## 📌 Overview
This project detects human emotions (Happy, Sad, Angry, Neutral) from facial expressions using deep learning.

## 🚀 Features
- Real-time face detection using OpenCV
- CNN-based emotion classification
- Accuracy: ~85%

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- OpenCV

## ▶️ How to Run
1. Install required libraries
2. Run main.py
3. Webcam will start detecting emotions
